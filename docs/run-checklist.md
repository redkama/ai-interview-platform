# 실행 체크리스트

- [ ] MySQL이 `3306` 포트에서 실행 중이다.
- [ ] 백엔드가 `8080` 포트에서 정상 기동된다.
- [ ] AI 서비스가 `8000` 포트에서 정상 기동된다.
- [ ] 프런트가 `5173` 포트에서 정상 기동된다.
- [ ] 샘플 사용자 `demo@aimentor.dev / password123`로 로그인된다.
- [ ] 영어/한국사 샘플 레슨이 화면에 보인다.
- [ ] 도서 목록, 장바구니, 주문 화면이 정상 표시된다.
- [ ] 면접 질문 생성 또는 스텁 면접 흐름이 정상 동작한다.
- [ ] 대시보드에서 차트, 진행률, 최근 주문이 보인다.
