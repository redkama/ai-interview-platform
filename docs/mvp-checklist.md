
# MVP 체크리스트

## 완료된 것
- 사용자 인증과 JWT 기반 로그인 흐름
- 이력서, 자기소개서, 채용공고 CRUD
- 면접 세션 시작, 답변 저장, 결과 리포트
- 영어 교육 도메인과 한국사 교육 도메인
- 도서 목록, 장바구니, 주문 기본 흐름
- FastAPI 기반 AI 마이크로서비스 구조
- 로컬 전용 샘플 데이터 시더
- 프런트 시연용 대시보드와 핵심 화면

## 미완료된 것
- 백엔드와 AI 서비스의 완전한 실연동
- 음성 인식/음성 평가 기능
- 대시보드 전용 집계 API
- 결제 연동과 주문 취소 정책
- 운영 DB 마이그레이션 정리

## 다음 작업
1. 백엔드에서 `AI_SERVICE_URL` 기반 HTTP 연동 추가
2. 프런트 스텁 데이터를 실제 API 응답으로 교체
3. 운영용 DB 마이그레이션 스크립트 추가
4. 대시보드 집계 API와 추천 로직 정리
5. 배포 환경 변수와 리버스 프록시 문서화
