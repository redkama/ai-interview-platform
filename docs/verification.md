# 검증 가이드

## 백엔드 빌드
```bash
cd backend
./gradlew build
```

## 백엔드 테스트
```bash
cd backend
./gradlew test
```

## 프런트 빌드
```bash
cd frontend
npm run build
```

## AI 서비스 실행 및 검증
```bash
cd ai-service
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

추가 확인:
- `http://localhost:8000/docs`
- `http://localhost:8000/health`

## 주요 수동 테스트 시나리오

### 회원가입
1. `/signup` 진입
2. 이름, 이메일, 비밀번호 입력
3. 가입 후 대시보드 이동 확인

### 로그인
1. `/login` 진입
2. 샘플 계정 또는 신규 계정으로 로그인
3. 인증 상태 유지 확인

### 이력서 등록
1. 이력서 페이지 진입
2. 이력서 생성
3. 목록과 상세 조회 확인

### 면접 세션 시작
1. 면접 설정 페이지 진입
2. 세션 시작 요청
3. 질문 목록 표시 확인

### 답변 저장
1. 질문별 답변 입력
2. 저장 버튼 클릭
3. 세션 상세 또는 결과 화면에서 반영 여부 확인

### 영어/한국사 학습
1. 영어 레슨 목록 조회
2. 영어 레벨 테스트 저장
3. 한국사 레슨 조회
4. 한국사 퀴즈 풀이와 오답 노트 확인

### 도서 장바구니/주문
1. 도서 목록 조회
2. 상세 화면에서 장바구니 담기
3. 장바구니 수량 변경
4. 주문 생성
5. 주문 목록과 상세 확인
